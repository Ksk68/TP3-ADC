**DASH** é uma plataforma colaborativa e inteligente desenvolvida em Portugal para revolucionar a gestão de marcações e agendamentos para clientes e estabelecimentos locais.

## 📋 Sobre o Projeto

O DASH foi concebido para resolver a fragmentação do mercado de agendamentos, unificando setores distintos como:
- 🏨 Alojamento
- 🍽️ Restauração
- 🎮 Lazer
- 💆 Bem-estar

Tudo isto debaixo de um único motor lógico inteligente.

> **Nota:** Este é um projeto académico desenvolvido como trabalho de estudos em Ambientes de Desenvolvimento Colaborativo.

## 🚀 Como Começar

### 1. Clonar o Repositório

```bash
git clone https://github.com/Ksk68/TP3-ADC.git
cd TP3-ADC/DASH
```

### 2. Instalar Dependências

```bash
npm install
# ou
yarn install
```

### 3. Iniciar o Servidor de Desenvolvimento

```bash
npm start
# ou
yarn start
```

Este comando inicia um servidor de desenvolvimento local e abre automaticamente a aplicação no seu navegador. A maioria das alterações são refletidas em tempo real sem necessidade de reiniciar o servidor.
