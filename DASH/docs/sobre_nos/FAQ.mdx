# ❓ Perguntas Frequentes (FAQ)

Consulte as respostas às dúvidas mais comuns sobre o funcionamento do ecossistema DASH.

---

### O DASH cobra alguma taxa aos clientes finais pelas reservas?
> **Resposta:** Não. Para os clientes, a utilização da aplicação para pesquisar e agendar serviços é totalmente gratuita.

### É possível integrar o DASH com bases de dados antigas de hotéis?
> **Resposta:** Sim. O nosso back-end foi desenhado em Python utilizando estruturas modulares, o que facilita a criação de scripts de migração de dados através de ficheiros JSON ou CSV.


:::warning[Aviso]
O DASH é uma empresa fictícia em que nem todas as ferramentas estão prontas 
para fase de lançamento.
:::
